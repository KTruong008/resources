# adv-todo

<p>A Todo App created with modern JS tools.</p>

<h4>Notes</h4>
<li>Coded sometime Summer or Fall of 2016</li>
<li>Uses Max Stoiber's react-boilerplate</li>

<li>Using Immutable was more verbose than expected</li>
